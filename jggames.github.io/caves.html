Caves of the Unwashed Heathen By Paul Shoemaker (2000s)

Main program files runable from emulator:
  • CAVESINN - In the emulator select CAVESINN from the Cassette menu. Then type RUN and hit Enter
  • CAVES - After running CAVESINN to create a character file, run this program to play that character
  • MAPMAKER - Program to make your own level maps
Download each of the following files before launching the Main program files:
  • LVL1DATA.C10 - Level 1 data file
  • LVL2DATA.C10 - Level 2 data file
  • LVL3DATA.C10 - Level 3 data file
  • LVL4DATA.C10 - Level 4 data file
Further Instructions:
This a homebrew RPG made long after the age of 8-bit home micros was over. It is an homage to the great RPGs of the early 80s.
Because of its size and complexity it takes multiple files to run, which means you must download those files to your computer and load them when prompted.
To save or load files use the "Start record" or "Choose File" buttons of the emulator when prompted. Additional information about the program can be found here.
