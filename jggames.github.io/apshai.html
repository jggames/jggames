Type-in Mania: An Interactive History of Early Basic Computer Applications

Temple of Apshai by Automated Simulations (1979)

Main program files runable from emulator:
  • INN - In the emulator select INN from the Cassette menu. Then type RUN and hit Enter
  • DM - After running INN to create a character file, run this program to play that character
Download each of the following files before running the Main program files:
  • LEVEL1.C10 - Level 1 data file
  • LEVEL2.C10 - Level 2 data file
  • LEVEL3.C10 - Level 3 data file
  • LEVEL4.C10 - Level 4 data file
Further Instructions:
This is one of the first commercially successful computer RPG programs. It was written in BASIC.
Because of its size and complexity it takes multiple files to run, which means you must download those files to your computer and load them when prompted.
To save or load files use the "Start record" or "Choose File" buttons of the emulator when prompted. A manual for the game can be found here.
