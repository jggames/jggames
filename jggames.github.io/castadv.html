Type-in Mania: An Interactive History of Early Basic Computer Applications

Castle Adventure by David Malmberg (1982)

Main program file runable from the emulator:
  • CASTADV1 - In the emulator select CASTADV1 from the Cassette menu. Then type RUN and hit Enter. Then load the data file below
Download the following file before launching the Main program file:
  • CASTADV2.C10 - Data file
Further Instructions:
This text adventure is very large and complex, so it takes multiple files to run. You must download the data file to your computer and load it when prompted.
To save or load files use the "Start record" or "Choose File" buttons of the emulator when prompted.
